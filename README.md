Student Social Media & Mental Health Analytics Dashboard
Project Overview

This project analyzes the relationship between social media usage and student mental health using Power BI.

The dashboard provides insights into:

Social media usage patterns
Mental health scores
Stress levels
Sleep quality
Study habits
Physical activity
Country-wise performance

Key Insights
Higher social media usage is associated with lower mental health scores.
Poor sleep quality increases stress levels.
Students with balanced study hours show better wellbeing.
Instagram and TikTok are the most frequently used platforms.
Physical activity positively influences mental health.
